# Changelog

All notable changes are documented here. Dashbored follows [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Added

- Public release workflow that publishes signed, multi-architecture container images to GitHub Container Registry on `v*` tags.
- Dependency update automation, container build validation, logical database backups, and a v0.1 support policy.

## [0.1.0] - 2026-07-18

### Added

- Private dashboard with encrypted credentials, PostgreSQL persistence, Better Auth, a scheduled worker, and Docker Compose deployment.
- Portainer inventory discovery, Launcher synchronization, and confirmed container actions.
- Public Mintlify documentation and an AGPL-3.0 license.
