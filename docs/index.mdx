---
title: What is Dashbored?
description: A calm, private front door for the services that run your home server.
---

Dashbored brings service health, practical controls, container discovery, and launch links into one local-first console. It is built for the person operating the infrastructure—not for a public status page or wall display.

## One operational view

Connect the services you already trust, then use the dashboard to see what needs attention before opening an individual application. Native integrations provide focused data; everything else can remain a lightweight Launcher entry.

## Private by design

Dashbored is intended for a LAN or private overlay such as Tailscale. Credentials are encrypted at rest, decrypted only server-side, and never returned to the browser.

## Supported integrations

- Portainer for discovery and confirmed container actions
- Beszel for host and container telemetry
- AdGuard Home for DNS activity and filtering health
- Uptime Kuma for monitor availability and latency
- Jellyfin and Immich are part of the supported-integration roadmap
